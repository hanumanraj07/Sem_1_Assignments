Getting Started

BNB Chain is EVM-compatible, making it easy for Ethereum developers to port their dApps. Follow these steps to start building on BNB Chain.

Set Up Your Development Environment

Install Node.js, a code editor, and set up MetaMask to connect to BNB Smart Chain.

npm install -g truffle

Configure MetaMask

Add BNB Smart Chain network to your MetaMask wallet with the following details:

Network Name: BNB Smart Chain
RPC URL: https://bsc-dataseed.binance.org/
ChainID: 56
Symbol: BNB
Block Explorer: https://bscscan.com

Get Testnet BNB

Visit the BNB Chain Faucet to get testnet BNB for development and testing.

Get Testnet BNB

Deploy Your First Contract

Use Truffle, Hardhat, or Remix to compile and deploy your smart contract to BNB Smart Chain.

truffle migrate --network bsc_testnet
